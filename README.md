## What's Inside

This repo has last-time exam prep material for Class 11th - basically the stuff I wish I had during my exams. Everything here is written by me, keeping both Hindi and English medium students in mind.

### Subjects Covered

- **Physics** - Notes, derivations, numericals, important questions
- **Chemistry** - Organic, inorganic, physical - all three covered
- **Mathematics** - Formulas, solved examples, practice problems
- **Hindi** - Grammar, essays, question banks
- **English** - Literature summaries, writing skills, grammar

All content available in both Hindi and English medium where applicable.

---

## How to Use

1. Check the subject folder you need
2. Start with "Last Minute Revision" files if exams are close
3. Go through custom books for detailed understanding
4. Practice numericals from solved examples
